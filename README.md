# M0 B2 — Intégrer une IA sur étagère (variante scikit-learn)

Dans ce module introductif, vous découvrez comment entraîner un modèle de classification simple avec **scikit-learn**, le sérialiser avec **Joblib**, puis l'intégrer dans une application complète : exposition via une API REST (FastAPI) et interface utilisateur (Streamlit). L'objectif est de comprendre le cycle complet d'une IA "sur étagère" : de l'entraînement au déploiement conteneurisé.

## Livrables
   - `M0_B2_Notebook.ipynb` : Notebook compte rendu du brief
   - `train_v1.py` : entrainement du modèle et production des fichiers joblib
   - `api_joblib.py` : Exposition via FastApi
   - `M0_B2_streamlit.py` : Interface utilisateur
